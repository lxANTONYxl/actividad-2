# Trabajo Grupal — Taller de Aplicaciones en Internet

Página web desarrollada de forma colaborativa como parte del curso de
Taller de Aplicaciones en Internet. El sitio presenta una estructura
completa con navegación, sección de servicios, información del equipo
y un formulario de contacto funcional conectado a un servidor HTTP
desarrollado en Python.

## Tecnologías utilizadas

- HTML5
- CSS3
- Python 3 (HTTPServer)
- GitHub (control de versiones)

## Estructura del proyecto
proyecto/
├── index.html
├── styles.css
└── python.py

## Ejecución

```bash
python python.py
```

Abrir en el navegador: http://localhost:8000
